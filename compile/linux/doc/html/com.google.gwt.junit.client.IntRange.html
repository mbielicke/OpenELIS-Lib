Class IntRange

public class IntRange
extends Object
implements Range
A Range that iterates over a start and end value by a stepping function. Typically used by benchmarks to supply a range of values over an integral parameter, such as size or length.

Constructors

IntRange(int, int, Operator, int)Creates a new range that produces Iterators which begin at start, end at end and increment by the stepping function described by operator and step.
